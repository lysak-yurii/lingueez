# Dictionary

A desktop app for collecting and studying vocabulary. Built with Python and PySide6.

## Features

- Add, edit and organize words with translations, tags and statuses
- Live search and filtering
- Text-to-speech and MP3 export
- Export to PDF, Excel and CSV
- Cloud sync

## Setup

    ./run.sh

The first run creates a virtual environment and installs the dependencies.
Requires Python 3.12.
